# EVE-O Preview (汉化增强版)

本项目是基于 [EveOPlus/eve-o-preview](https://github.com/EveOPlus/eve-o-preview) 的汉化与功能增强分支。

EVE-O Preview 是一款为《星战前夜》(EVE Online) 玩家设计的轻量级多开窗口管理工具。它可以为每个后台运行的 EVE 客户端生成实时缩略图，帮助多开玩家轻松地在多个屏幕之间监控和快速切换不同的角色窗口。

## ✨ 本分支特色功能

除了拥有原版的所有强大功能外，本分支特别添加了以下增强特性：

* **全界面中文汉化**：由 AMIYA-老余 提供汉化，更符合中文玩家的使用习惯，降低使用门槛。
* **客户端自定义备注 (新增功能)**：
    * 支持为每个单独的游戏客户端（角色）添加个性化备注（例如：“拉货号”、“旗舰”、“化矿”等）。
    * 备注信息会直接显示在预览悬浮窗的标题栏中（格式如：`角色名 [备注]`），让你在十几开时也能一眼认出每个窗口的作用！

## 🚀 核心原版功能

* **实时缩略图预览**：监控所有后台客户端的实时画面。
* **画中画悬浮**：缩略图可自由拖拽、缩放、并保持在其他窗口顶部。
* **快捷键与循环组**：支持为特定客户端绑定快捷键，或使用快捷键在一组客户端之间快速前后切换。
* **布局记忆**：自动记忆不同客户端的缩略图位置，下次启动直接恢复。
* **性能友好**：极低的 CPU 和内存占用。

## 📖 如何使用“自定义备注”功能？

1. 启动 EVE-O Preview 并在主界面左侧的 **“活动客户端”** 列表中选中一个角色。
2. 在列表下方的 **“备注”** 输入框中，填写你想给该角色标记的文字。
3. 鼠标点击软件空白处（或切换输入焦点），程序会自动保存备注，并且悬浮窗将自动刷新显示你的备注。
